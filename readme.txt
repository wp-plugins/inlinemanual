Connect Inline Manual with your Wordpress.

== Description ==

Inline manual is an online documentation tool, making it easy for everyone to create, update and re-use documentation for your WordPress end-users. Teach them how to use the website you have just created through in-app walkthroughs, tooltips and topic launchers.

== Installation ==

1. Launch the "add plugin" page in your WP admin.
2. Find the "Inline Manual" plugin
3. Activate the plugin
4. The tour should guide you through

== Frequently asked questions ==

= Do I require to have Inline Manual account? =

Yes and No, you need the API key for your site, which you can get from the InlineManual.com portal if you want to have your own walkthroughs.

= Is it production ready? =

Yes, the plugin is production ready.
