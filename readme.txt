Connect your "Inline Manual" with your Wordpress.

== Description ==

Inline manual is an online documentation tool, making it easy for everyone to create, update and re-use documentation for your WordPress end-users. Teach them how to use the website you have just created.

Inlinemanual plugin lets you connect to your <a href="http://inlinemanual.com">InlineManual.com</a> account where you can fetch your manuals (topics). Each topic shown to users based on their capabilities (Hidden, Public, Subscriber, Contributor, Author, Editor, Administrator).

== Installation ==

1. Launch the "add plugin" page in your WP admin.
2. Find the "Inline Manual" plugin
3. Activate the plugin
4. Fill in your API key in the WP Admin > Inlinemanual > account (tab)
5. Fetch your topics, set the permissions
6. Enjoy

== Frequently asked questions ==

= Do I require to have Inline Manual account? =

Yes, you need the API key for your site, which you can get from the InlineManual.com portal.
